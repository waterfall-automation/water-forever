# Water Forever — Digital Water Infrastructure for Generations

Water Forever is a marketing and awareness initiative focused on building a digital intelligence layer for global water infrastructure—the **Waterfall Grid**. 

By connecting reservoirs, aquifers, treatment plants, agricultural irrigation sensors, and municipal networks, Water Forever advocates for a unified, open, and responsive water intelligence ecosystem.

## Project Structure
- `index.html`: A single-page static marketing website featuring:
  - An interactive, mouse-responsive HTML5 Canvas background wave animation.
  - Generous editorial typography and layout inspired by Suntory's "Passion for Water" aesthetics.
  - A scroll-triggered timeline and pressure points dashboard.
  - An interactive SVG "Waterfall Grid" architecture showing data and water flows.
  - A direct email client mailto integration to Join the Initiative.
- `Water Forever - The founding document.pdf`: The baseline strategic document outlining the initiative's mission and architecture.

## Visual and Technical Design
- **Editorial Typography**: Styled using `Fraunces` (serif) and `Inter` (sans-serif) for high readability and premium aesthetic.
- **Subtle Motion**: Leverages lightweight, native CSS path morphs and canvas waves to minimize processor usage.
- **Interoperability-first**: Pure HTML, CSS, and JS with zero third-party framework dependencies.
